Тэнгэрийн Мэлмий — Мөнгөний блок V8 Firebase

Шинэ боломж:
- Firebase Firestore холболттой.
- Захиалга orders collection-д хадгалагдана.
- Админ самбар Firebase-аас захиалга шууд уншина.
- Тайлан үүсгээд reports collection-д хадгална.
- Хүн бүрт өвөрмөц линк үүснэ: /#report=REPORT_ID
- SMS линк илгээх товч нь утасны SMS app нээнэ.
- PDF татах, хэвлэх, текст хуулах хэвээр.

Админ нууц үг: 9999

Firebase шаардлага:
- Project: mongo-block
- Firestore collection: orders үүссэн байх.
- Test mode эсвэл web read/write зөвшөөрсөн rules байх.

GitHub-д upload хийхдээ ZIP өөрийг нь биш, доторх бүх файлыг root дээр тавина.
